  • Zodiac Finder

    Trying to get into daily horoscope readings but forgot your zodiac sign? Fear not! Simply select your birthday from the dropdown lists for month and day, and retrieve your zodiac name, image, information, and associated constellation. This information may or may not guarantee accurate predictions of your future.

    Notes: The number of days changes based on month selection through the appendChild function. This function is controlled through the use of two global boolean variables whose values create unique conditions for each of three types of months: months with 31 days, months with 30 days, and a month with 29 days. Depending on the initial value of these booleans on month selection, the correct number of days will be appended or removed accordingly.

  • Who's Paying?

    Have you ever gone out to dinner with friends? Have you ever had to deal with everyone swiping out their credit cards only for your waiter to inform you that they cannot split the bill? Regardless of your answer, this situation is inevitable, but luckily for you, there is a solution. This application prompts you to specify what your group is paying for. Then, simply type in everyone's names to add/remove them from a list, and hit a button to select a random person from this list who will then be required to pay on behalf of the group. You're welcome.

    Notes:Typing in a name pushes that name into an array (and displays it in a list), unless the name already exists in the array. User input is converted into a first-character-capitalized-and-rest-lowercase format. To remove a name from the array, user input must match an existing name in the array (not case sensitive). Clicking "Choose Random Name" executes a Math.floor(Math.random()*array.length) function that generates a random array index number. The name at this random index is then displayed within a div. Clicking outside this div will cause the results div to disappear.

  • Hogwarts Character Creator

    Although the Harry Potter book series has come to an end, we all know that the Harry Potter Universe far from dead. Whether you're an average Harry Potter nerd, not that into Harry Potter, or still writing Harry Potter fanfiction after all these years (that is dedication), you might enjoy passing your time with this fun little character generator. The character creator has just enough controllable elements and random elements to make it interesting enough to make several characters, whether you plan on using them in your fanfiction or not.

    Notes:User input for all questions is gathered and displayed in results. For all questions excluding numbers 3-5, the input directly determines the value of the results. In question 3, users select one of three animal familiars. For each type of familiar (toad, cat, owl), there are five possible breeds, each with its own image attached. Based on user input for this question, the five breeds associated with the animal selection are pushed into an array, and a random one is selected to display in the results. Wand core is determined by question 4, in which each trait corresponds with one of three core types. Wand wood is determined using official Pottermore data; for each possible combination of eye color and trait most proud of, there are three possible wood types. These three wood types are pushed into an array, and the results display a random wood from this wood Array. The inputs for First Name, Last Name, and email address are controlled by regular expressions. Not satisfying the expression terminates the form submit function and displays an error message in the bottom status box.

  • Student Raffle

    During my time as director of a middle-school math mentoring program, my team created a "sticker board" in a desperate attempt to help motivate students to work on math while their peers played outside. The deal was that each sticker would be converted into a raffle ticket at the end of the semester that would be put into a drawing for prizes. The names have been removed, but this application shuffles all the students' "tickets" and draws them one by one.

    Notes: Each student's name was inserted into an array for however many stickers they had. For example, if Student Z earned x number of stickers, their name was inserted into the array x number of times. The names are randomly shuffled into a new array, which is then printed one by one (to mimic a raffle without replacement).
